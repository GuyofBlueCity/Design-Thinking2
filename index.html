Screen 1 of 3 — Building Join
Screen annotations
1
Neighbour count & avatars — showing how many residents have already joined addresses the critical-mass problem surfaced in peer testing. If the number is too low, users won't join. Tests A2: Neighbour trust
2
Verification trust badge — explains that residents are verified via building entry code. Tests whether visible safety signals lower the social barrier to joining strangers' food-sharing community. Tests A2: Trust between strangers
3
Real initials on avatars — showing real neighbour initials (not generic icons) creates a sense of real community identity before the user commits to joining. Tests A2: Social comfort
Screen annotations
1
Barcode scan as primary CTA — placed above manual entry to test whether scan-first reduces friction enough for users to log items. Peer testing showed manual typing causes immediate drop-off. Tests A1: Will users actually log items?
2
"Auto-detected from barcode" tag — signals to users that the app did the work. Tests whether perceived low effort is sufficient to sustain daily or weekly logging habits. Tests A1: Logging sustainability
3
Prominent expiry date field — pre-filled and highlighted. Tests whether users confirm or ignore auto-detected expiry dates — the quality of this data determines the whole notification system. Tests A5: Expiry awareness
Screen annotations
1
Expiry alert with inline Share CTA — the core test for A5: does pairing an expiry notification directly with a one-tap "share with building" action produce different behaviour than a bare reminder? Tests A5: Notifications → action
2
Red/amber/green expiry badges — urgency-coded visual system. Tests whether colour-coded UI cues cause users to prioritise consumption or sharing without needing to read dates carefully. Tests A5: Visual urgency nudges
3
Demand signal nudge card — "3 neighbours want Dairy items" shows real community pull. Tests whether social demand signals increase sharing likelihood beyond what a plain notification would achieve. Tests A2: Community engagement
